# 英语背单词网站需求分析

## 1. 用户角色

### 1.1 教师用户
- 拥有最高权限
- 可以管理词书和单词
- 可以创建和管理班级
- 可以查看所有学生的学习进度

### 1.2 班级用户（学生）
- 可以学习教师分配的词书
- 可以参与单词游戏
- 可以查看自己的学习进度

## 2. 初次部署流程

### 2.1 首次访问设置
- 系统检测是否已设置教师用户
- 如果未设置，引导用户创建教师账户
- 教师账户创建后，系统初始化基本配置

## 3. 词书管理

### 3.1 自动爬取词书
- 支持从指定网站自动爬取词书
- 爬取的词书信息包括：词书名称、描述、单词列表等

### 3.2 教师选择词书
- 教师可以从已爬取的词书中选择合适的词书
- 教师可以为不同班级分配不同的词书

### 3.3 自定义录入单词
- 教师可以手动添加单词到词书中
- 支持批量导入单词

## 4. 界面要求

### 4.1 类似百词斩的样式
- 简洁清新的界面设计
- 卡片式单词展示
- 图片与单词结合
- 进度可视化展示

## 5. 功能要求

### 5.1 单词游戏功能
- 提供多种单词游戏模式
- 支持闯关模式
- 记录游戏得分和学习进度

## 6. 部署要求

### 6.1 部署方式
- 不使用Docker容器化部署
- 支持传统服务器部署
- 提供详细的部署文档
